# 🚨 BRAHMASTRA  
### Women Safety & Emergency Response System

> *“HER safety belongs in HER hands.”*

---

## 💡 Overview

*Brahmastra* is a powerful women safety application designed to provide *instant emergency response, real-time support, and quick access to help* in critical situations.

Built during a hackathon, this project focuses on *speed, simplicity, and life-saving actions* when every second matters.

---

## ⚡ Key Features

### 🚨 Emergency SOS
- One-tap *SOS button*
- Sends alerts to all emergency contacts
- Quick access to police (100) and ambulance (102)

---

### 📍 Live Location Tracking
- Real-time location sharing
- Helps contacts track user instantly
- Integrated map-based visualization

---

### 📞 Emergency Contacts
- Add, manage, and trigger SOS for contacts
- One-click alert to all saved contacts

---

### 🔊 Safety Tools
- Loud alarm to attract attention
- Fake call feature for escape situations

---

### 📝 Voice & Text Notes
- Record voice notes during emergencies
- Save important details instantly

---

## 🛠️ Tech Stack

*Frontend:*
- React.js
- Modern UI with custom styling

*Backend:*
- Node.js
- Express.js

*Database:*
- MongoDB (Atlas)

*Authentication:*
- JWT (JSON Web Tokens)

*Deployment:*
- Render

---

## 🔐 Environment Variables

Create a .env file in backend:
```
JWT_SECRET=your_secret_key
 MONGO_URI=your_mongodb_connectio
```

## 🚀 Getting Started

### 1️⃣ Clone the repository
https://github.com/priyakalamkar555-art/BRAHMASTRA
---
### 2️⃣ Install dependencies

#### Backend:
```
cd server
npm install
```
#### Frontend
```
cd client
npm install
```

---

### 3️⃣ Run the project

#### Backend:
```
npm start
```
#### Frontend
```
npm start
```
